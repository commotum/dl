# downloadkit TODO

## Goal

Build `downloadkit` as the lean file transfer layer for personal use:

- robust single-file downloads
- resumable downloads
- atomic writes
- fallback URLs
- response/content validation

This should be the layer that handles "take a URL and make the file land correctly on disk."
