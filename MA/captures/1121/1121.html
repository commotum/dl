Introduction

Probability theory is about repeatable processes whose outcomes cannot be predicted with certainty. These processes are called experiments or trials.

  • For instance, tossing a coin is a probability experiment because we don't know beforehand whether the coin will land on heads or tails.

  • Likewise, rolling a die is a probability experiment because we don't know beforehand what number the die will land on.

The set of all the possible outcomes of an experiment is called the sample space and is usually denoted by Any subset of the sample space is called an event.

  • In the experiment of tossing a coin, the only possible outcomes are (heads) and (tails). So, the sample space is An example of an event is which refers to the event that the coin lands on heads.
  • In the experiment of rolling a die, the possible outcomes are numbers through So, the sample space is An example of an event is which refers to the event that an odd number is rolled.

Lastly, an experiment is said to be deterministic if it has only one possible outcome. For example, if a bowl contains apples and nothing else, and the experiment involves picking an item from the bowl, then the only possible outcome is that we pick an apple. So, this experiment is deterministic.

Deterministic experiments are not probability experiments. Unlike probability experiments, we can predict deterministic experiments with certainty.

  • II is not a probability experiment because it is deterministic. No matter how many times we count, we will get the same outcome each time.

  • III is a probability experiment. Each time we repeat the experiment, we can select different children and get different weights.

In conclusion, the correct answer is "III only."
