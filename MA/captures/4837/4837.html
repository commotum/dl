Binary relations can be visualized using a plot (or graph).

For example, consider the binary relation defined over and given by

The relation corresponds to the plot shown below, where

  • the elements from the first set are written along the horizontal axis,

  • the elements from the second set are written along the vertical axis, and

  • each pair defines a point with coordinates on the plot.

Plots usually work well for finite and infinite (discrete or continuous) relations.

Watch out! A relation may have points that line on the same vertical line, like the points and in In general, a relation is not a function. Functions are a special case of relations, where the points with the same first component but different second components are not allowed. We'll learn more about these special kinds of relations in future lessons.
