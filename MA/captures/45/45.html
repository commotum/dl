Introduction
?

A set is a collection of objects known as elements or members.

Usually, the elements of a set have some property in common. For example, a set could be a class at school where the students are the members of the set. The common property is that all the students are members of the class.

To write a set, we list the elements between two braces (curly brackets). Commas are used to separate the elements.

So, for example, the set of the primary colors would be written as
