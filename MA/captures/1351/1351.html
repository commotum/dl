A Summary of Image Types
?

In general, for any linear transformation with associated matrix we have one of the following three possibilities:

  • If is non-singular, i.e., then the image of a two-dimensional object is a two-dimensional object. In particular, the image of a polygon with vertices is a polygon with vertices.

  • If is singular, i.e., we have two additional subcases:

    • If has at least one nonzero entry, then the image of a two-dimensional object is a one-dimensional line segment.

    • If is the zero matrix, then the image of a two-dimensional object is the point the origin of the coordinate system.
