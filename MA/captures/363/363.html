The concavity of a function refers to the direction in which its graph curves.

  • If the graph curves upwards, like a smile, then the graph is said to be concave up ().

  • If the graph curves downwards, like a frown, then the graph is said to be concave down ().

Try to remember the phrase "concave down like a frown."
