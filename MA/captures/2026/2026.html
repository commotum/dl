Before we move on, note the following:

  • The fulcrum also has mass. However, since the distance between the fulcrum and itself is zero, its moment about the fulcrum is zero, and it doesn't feature when we apply the principle of moments. Thus, taking moments about the fulcrum was convenient because we don't need to know its mass! Generally, taking moments about a convenient point can save us a lot of work.

However, we'll soon see that selecting the signs using this convention makes it easy to model moment problems using coordinate axes.
