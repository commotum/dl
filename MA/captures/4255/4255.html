Example: Using Laws Involving Conjunction and Disjunction
?

Write a sentence that is equivalent to the following: " is an integer, and is negative and it is an integer."

EXPLANATION

Using the commutative law, we can replace the phrase " is negative and it is an integer" with the phrase " is an integer and it is negative." The sentence now reads

" is an integer, and is an integer and it is negative."

Then, using the associative law, we can move the comma:

" is an integer and is an integer, and it is negative."

Now, we can simplify the phrase " is an integer and is an integer" to just " is an integer" The sentence now reads

" is an integer, and it is negative."

Finally, we can write the sentence more concisely:

" is a negative integer."
