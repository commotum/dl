For example:

  • Any integer is congruent to itself modulo for any natural So, congruence of integers is reflexive.

  • The relation "are born in the same year," defined on the set containing the entire human population, is also reflexive. This is because any person was born in the same year as themselves.

A binary relation on a set is called irreflexive if for every element In other words, no element of is related to itself by

For example:

  • The relation "is less than" defined on the set of real numbers, is irreflexive since no number is less than itself. In other words, the statement is false for any
  • The relation "is the sisters of," defined on the set containing all people, is irreflexive. This is because a person can't be their own sister.
